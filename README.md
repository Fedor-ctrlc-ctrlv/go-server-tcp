# go-server-tcp

# TCP Чат сервер на Go
Многопользовательский чат-сервер, написанный на чистом Go с использованием TCP сокетов. Поддерживает комнаты, приватные сообщения и множество одновременных подключений.

## Технологии
- **Go 1.21+** - Основной язык разработки
- **Net package** - Работа с TCP сокетами
- **Goroutines** - Легковесные потоки для клиентов
- **Sync package** - Мьютексы для конкурентного доступа

### Требования
- **Go 1.21 или выше**
- **Telnet или netcat для подключения (опционально)**

## Что реализовано:
- **Многопользовательский чат** - Поддержка сотен одновременных подключений
- **Система комнат** - Создание и переключение между комнатами
- **Приватные сообщения** - Личные сообщения между пользователями
- **Список пользователей** - Просмотр участников в комнате
- **Системные уведомления** - Оповещения о входе/выходе пользователей
- **Масштабируемость** - Каждый клиент в отдельной горутине
- **Потокобезопасность** - Использование мьютексов для синхронизации
- **Протокол команд** - Удобные текстовые команды
